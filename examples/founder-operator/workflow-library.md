# Example Founder/Operator AI Workflow Library

This is a fictionalized example.

## Weekly strategy review

### Area

Strategy

### Use case

Review the current operating profile, priorities, risks, and open decisions.

### Why it matters

Keeps AI recommendations tied to current operating reality instead of stale assumptions.

### Reusable prompt

"Using my current operating profile and this week's priorities, identify the top strategic decisions I need to make, the risks I am underweighting, and the highest-leverage next actions. Separate facts, assumptions, and unknowns."

### Inputs needed

- Current goals
- Active projects
- Metrics
- Open decisions
- Constraints

### Output format

- Top 5 decisions
- Key risks
- Recommended next actions
- Questions that would materially change the plan

### Risk or failure mode

The model may over-index on stale context or generic founder advice.

### Recommended frequency

Weekly
