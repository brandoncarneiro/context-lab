# Example Memory Cleanup

This is a fictionalized example.

## Should be saved as durable memory

- "The user prefers direct, execution-focused advice that separates facts, assumptions, and unknowns."
- "The user works on practical AI workflow systems and values inspectability over hype."

## Should be kept only in this project/context document

- Current quarterly goals
- Active repositories under development
- Launch plans
- Sales pipeline
- Specific investor or customer conversations

## Should not be saved because temporary

- Weekly priorities
- Draft messaging
- Active task lists
- One-off travel plans

## Should be deleted or corrected

- Any old role, company, or project status that is no longer current

## Sensitive context to use only when explicitly provided

- Customer names
- Deal terms
- Revenue numbers
- Private team issues
