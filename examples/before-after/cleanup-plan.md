# Example Cleanup Plan

This is a fictionalized cleanup plan.

## Should be saved as durable memory

- "The user prefers direct, execution-focused advice."
- "The user values separating confirmed facts, reasonable inferences, and unknowns."
- "The user prefers recommendations that include risks, tradeoffs, and next actions."

## Should be kept only in this project/context document

- Current quarterly goals
- Active projects
- Current launch plan
- Current tool stack if it is changing
- Current operating cadence

## Should not be saved because it is temporary

- This week's priorities
- Draft messaging
- Active task lists
- Short-term travel or scheduling constraints

## Should be deleted or corrected if currently remembered

- "The user is fundraising this quarter."
- "The user is focused on the Acme partnership."
- "The user prefers aggressive growth strategy."
- Any old role, company, or project status that is no longer current

## Sensitive context that should only be used when explicitly provided

- Customer names
- Deal terms
- Revenue numbers
- Private team issues
- Personal health or family details
