# Context Hygiene

Context hygiene is the practice of keeping AI context accurate, current, scoped, and useful.

## Good context is

- dated
- specific
- source-aware
- easy to update
- separated by stability
- clear about uncertainty

## Bad context is

- stale
- vague
- flattering
- overconfident
- too tactical
- copied blindly into memory

## Context review cadence

Review your operating profile:

- monthly for high-change work
- quarterly for stable work
- immediately after role, organization, project, or personal constraint changes

Review memory:

- when ChatGPT gives odd personalized advice
- after major life or work changes
- before relying on it for serious strategy
