# Security Policy

Context Lab is a Markdown-only workflow repository. It does not run model calls, collect user data, store memory, or connect to ChatGPT accounts.

## Reporting

Use GitHub private vulnerability reporting when available. If private reporting is unavailable, open a minimal public issue that does not include private memory content, account details, credentials, private URLs, or sensitive examples.

## Sensitive Context Boundary

Do not commit or post:

- real saved-memory exports
- private operating profiles
- active customer, investor, hiring, medical, financial, or family context
- account screenshots
- credentials, tokens, cookies, or private URLs
- examples that can identify a real person without consent

Public examples in this repository should stay fictionalized, synthetic, or fully scrubbed.

## Supported Versions

Security fixes target the current `main` branch until versioned releases exist.
