# 06 Master Prompt

## Purpose

Use this last to create reusable prompts for new ChatGPT conversations.

## Prompt

Create a reusable master prompt I can paste into a new ChatGPT conversation whenever I want high-quality help.

The prompt should tell ChatGPT:

- Who I am
- What I am working on
- How I make decisions
- How I want advice
- What context matters
- What assumptions not to make
- How direct or critical to be
- How to structure answers
- When to ask questions versus make a best-effort recommendation
- How to handle uncertainty
- How to optimize for execution

Make it concise enough to be practical, but detailed enough to materially improve output quality.

Also give me:

1. A short version under 1,500 characters.
2. A longer version for important strategy conversations.
3. A version optimized for writing help.
4. A version optimized for business strategy and execution.

## Expected output

- A practical master prompt
- A short prompt under 1,500 characters
- A strategy version
- A writing version
- A business execution version

## Failure modes

- Making the prompt too long for normal use
- Including tactical context that will stale quickly
- Forgetting to include uncertainty handling
- Asking the model to be contrarian without calibration
