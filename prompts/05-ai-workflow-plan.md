# 05 AI Workflow Plan

## Purpose

Use this after the operating profile and memory cleanup to turn context into practical, repeatable AI workflows.

## Prompt

Now turn this context audit into a practical AI workflow plan.

Identify the highest-leverage ways ChatGPT should help me across:

- Strategy and planning
- Sales, GTM, fundraising, or finance, if relevant
- Team, collaborators, hiring, or management, if relevant
- Product, operations, study, creative work, or delivery
- Writing and communication
- Research and decision support
- Personal productivity
- Health, fitness, travel, household, or personal planning, if relevant

For each area, give me:

1. Recommended use case
2. Why it matters
3. Example prompt I should reuse
4. Inputs ChatGPT would need from me
5. Output format I should ask for
6. Risk or failure mode
7. How often I should run it

Then rank the top 10 workflows by usefulness.

## Expected output

- A practical workflow library
- Reusable prompts
- Required inputs for each workflow
- Output formats
- Risks and run cadence
- A top 10 usefulness ranking

## Failure modes

- Producing abstract use cases with no reusable prompts
- Ignoring risks and failure modes
- Recommending workflows that depend on stale context
- Overfitting to one role or current project
