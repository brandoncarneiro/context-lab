# 03 Operating Profile

## Purpose

Use this after answering the gap interview to create a clean current operating profile.

## Prompt

Using my answers and your prior audit, create a clean "Current Operating Profile" for me.

Organize it into these sections:

1. Identity and role
2. Company and business context
3. Current goals
4. Current projects
5. Key metrics and outcomes
6. Decision-making style
7. Communication preferences
8. Tools and workflows
9. Personal context that is useful for recommendations
10. Constraints, risks, and blind spots
11. Things you should avoid assuming
12. Things that are outdated or uncertain

For each section, separate:

- Confirmed facts
- Reasonable inferences
- Unknowns

Then give me:

1. A concise version suitable for Custom Instructions.
2. A detailed version suitable for a saved context document.
3. A list of specific memories I may want ChatGPT to save.
4. A list of memories or assumptions I should delete or avoid saving.

## Expected output

- A dated operating profile
- Confirmed facts separated from inferences and unknowns
- A short Custom Instructions version
- A detailed saved-context version
- Memory recommendations and deletion recommendations

## Failure modes

- Mixing current tactical state with durable identity
- Treating guesses as confirmed facts
- Producing a master prompt too early
- Saving too much to memory
